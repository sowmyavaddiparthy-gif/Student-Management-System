# Student Management System

A simple and responsive Student Management System built using HTML, CSS, and JavaScript.
This project allows users to add, edit, delete, and store student records using Local Storage.



 # Features

-  Add new student details
-  Edit existing student data
-  Delete student records
-  Data stored using Local Storage (persists after refresh)
-  Dark Mode toggle
-  Responsive UI with sidebar navigation
-  Clean and modern design



 # Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

# How It Works

- Data is entered through a form
- On clicking Save, data is:
  - Added to the table
  - Stored in Local Storage
- On page reload:
  - Data is automatically loaded from Local Storage
- Edit and Delete buttons modify the table dynamically



# Dark Mode

- Toggle switch enables dark theme
- UI updates instantly
